Featured Projects

Selected Research and Engineering Work

Focus-Stack DFF Surface Defect Reconstruction

Focus-stack imaging and depth-from-focus pipeline for surface defect visualization, with glare-aware reconstruction, synthetic surface simulation, and real-sample relative reconstruction.

Depth-from-Focus / OpenCV / PyTorch / Simulation

Read more

This project explores multi-material surface defect visualization using focus-stack imaging, DFF reconstruction, glare-risk modeling, and learning-based depth correction. The public repository includes reproducible scripts for synthetic surface generation, focus-stack simulation, glare-aware focus evaluation, and representative comparison figures.

  • Built the Python pipeline for synthetic surface generation, focus-stack simulation, DFF reconstruction, and comparison figure generation.
  • Implemented the project-wide focus-stack height convention and ran algorithm comparisons across traditional and learning-based methods.
  • Prepared presentation-derived result panels with explicit boundaries between real-sample relative reconstruction and simulated quantitative validation.
View repository

Solar Cell Panel Crack Detection

Machine-vision workflow for crack-like defect detection in solar cell images, combining background correction, SVD reconstruction, morphology, GUI inspection, and a UNet segmentation workflow.

OpenCV / SVD / Tkinter / UNet

Read more

The project combines illumination correction, grid rectification, SVD-based reconstruction, thresholding, morphology, connected-component filtering, and skeleton/overlay visualization. A secondary workflow uses manually annotated SVD images to train a lightweight UNet model for pixel-level crack highlighting.

  • Integrated the traditional image-processing workflow for defect enhancement and candidate filtering.
  • Developed Tkinter GUI tools for parameter tuning, stage-by-stage visualization, and result export.
  • Added batch processing, HTML report generation, and a segmentation submodule for annotation, training, validation, and overlay inference.
View repository

Depth Ratio Detector

Image-analysis utility for measuring circular pore radii from microscopy-style image pairs and calculating depth ratio through OpenCV detection, manual least-squares fitting, and a Tkinter interface.

Circle Detection / Least Squares / OpenCV / Tkinter

Read more

Depth Ratio Detector is an engineering utility for measuring circular pore radii from top/bottom surface image pairs and calculating the sample depth ratio. It supports automatic Hough-circle detection, manual least-squares fitting, annotated result export, and repeated parameter presets for different imaging conditions.

  • Implemented the image-processing workflow for circle detection and difficult-image manual fitting.
  • Built the depth-ratio calculator interface and saved representative radius measurements for comparison.
  • Organized reusable scripts, parameter presets, representative images, and result samples for portfolio review.
View repository
